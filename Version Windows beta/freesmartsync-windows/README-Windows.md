# FreeSmartSync v0.8.6.2 — Guide Windows

## Prérequis

### 1. Python 3.10+
- Téléchargez : https://www.python.org/downloads/
- ⚠️ Cochez **"Add Python to PATH"** lors de l'installation

### 2. PyQt5 (installé automatiquement)
```cmd
pip install PyQt5
```

### 3. ADB (Android Debug Bridge)
- Téléchargez **Android Platform Tools** : https://developer.android.com/studio/releases/platform-tools
- Extrayez dans `C:\Android\platform-tools\`
- Ajoutez `C:\Android\platform-tools\` à votre **variable PATH**

**OU** copiez `adb.exe` directement dans le dossier `tools\` de FreeSmartSync.

## Installation

Double-cliquez sur **`INSTALLER-Windows.bat`**

## Lancement

- Double-cliquez sur **`Lancer-FreeSmartSync.bat`**
- Ou le raccourci créé sur le Bureau

## Activer le débogage USB sur votre téléphone

Le wizard vous guide au premier lancement selon votre marque de téléphone.

## ⚠️ Compatibilité

FreeSmartSync Windows utilise le même code que la version Linux.
Les fonctionnalités sont identiques.
